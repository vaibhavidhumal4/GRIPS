# The Sparks Foundation: GRIP 

## Overview
A collection of applied analytical projects developed during **The Sparks Foundation's Graduate Rotational Internship Program (GRIP)** in the Data Science & Business Analytics domain. 

These Jupyter Notebooks demonstrate end-to-end data workflows, focusing on extracting actionable business insights and building reliable predictive models using standard industry libraries.

## Completed Tasks

* **Task 1: Prediction using Supervised ML**
  * Developed a Simple Linear Regression model to predict a student's percentage based on the number of study hours.
* **Task 2: Prediction using Unsupervised ML**
  * Applied K-Means clustering to the Iris dataset to determine the optimum number of clusters and visualized the spatial distribution.
* **Task 3: Exploratory Data Analysis - Retail / Terrorism**
  * Performed extensive EDA on large-scale datasets to identify weak business areas, security trends, and correlations to assist in strategic decision-making.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
